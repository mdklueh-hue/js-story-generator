

# JavaScript Story Generator

Der Story Generator erstellt kurze Geschichten, indem vorderfinierte 
Wörter (z.B. Adjektive, Nomen und Verben) in einen festen Text eingesetzt werden.

## Projektziel

Ziel dieses Projekts ist es, grundlegende JavaScripts-Konzepte zu lernen 
und praktisch anzuwenden. Der Fokus liegt nicht auf Komplexität, 
sondern auf Verständlichkeit und sauberem Code.

## Was dieses Projekt zeigt

- Arbeiten mit Variablen (`let`)
- Umgang mit Strings
- String-verkettung mit dem `+`- Operator
- Wiederverwendung von Variablen mit neuen Werten
- Ausgabe von Ergebnissen über `console.log`
- Sinnvolle Kommentare zur Erklärung des Codes

## Funktionsweise

1. Wörter werden in Variablen gespeichert (z.B. Adjektive, Nomen, Verben).
2. Diese Variablen werden in einen festen Text eingesetzt.
3. Dadurch entsteht automatisch eine kleine Geschichte.
4. Durch das Ändern der Variablenwerte wird eine neue Geschichte erzeugt,
   ohne den restlichen Code zu verändern.

## Ausführen des Codes

Der Code kann z.B. mit Node.js ausgeführt werden:

node script.js
